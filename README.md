# Node.js Backend Project

## Description
This is a backend project built using Node.js and Express.js.

## Features
- User registration
- Login authentication
- Image upload using Multer
- JWT authentication

## Technologies Used
- Node.js
- Express.js
- MongoDB
- Mongoose

## Installation
1. Clone the repository
2. Run npm install
3. Create .env file
4. Run npm start

## API Routes
- POST /register
- POST /login
- GET /profile
